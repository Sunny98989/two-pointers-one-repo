# Express as Consecutive Number Sum

**Difficulty:** Easy
**Accuracy:** 50.49%
**Submissions:** 12K+
**Points:** 2

## Problem Statement

Given a number `n`, find whether `n` can be expressed as sum of two or more consecutive positive numbers.

---

## Examples

### Example 1

**Input:**

```text
n = 10
```

**Output:**

```text
true
```

**Explanation:**

```text
10 can be expressesd as 1 + 2 + 3 + 4.
```

---

### Example 2

**Input:**

```text
n = 8
```

**Output:**

```text
false
```

**Explanation:**

```text
8 can't be expressesd sum of two or more consecutive numbers.
```

---

### Example 3

**Input:**

```text
n = 24
```

**Output:**

```text
true
```

**Explanation:**

```text
24 can be expressesd as 7 + 8 + 9.
```

---

## Constraints

```text
1 ≤ n ≤ 10^9
```
